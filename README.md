# SentraTech Institute Landing Page

This is the official landing page for SentraTech Institute, a modern platform designed to showcase the institute's portfolio, services, and community initiatives.

## Features

- **Modern Tech Stack**: Built with Next.js 14 and React 18.
- **Responsive Design**: Fully responsive layout using Bootstrap 5 and custom SCSS.
- **Animations**: Smooth, high-performance animations powered by GSAP.
- **Dynamic Content**: Features dynamic sliders with Swiper and interactive elements.
- **Forms**: Robust form handling with React Hook Form.

## Landing Page Sections

The landing page is composed of several key sections:

- **Hero Section**: Captivating introduction with a main call-to-action.
- **Skills Section**: Showcases the institute's technical expertise and focus areas.
- **Services Section**: Details the services offered by SentraTech.
- **Portfolio Section**: Highlights recent projects and success stories.
- **Community CTA**: Encourages users to join the SentraTech community.

## Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/)
- **Library**: [React 18](https://react.dev/)
- **Styling**: [Sass (SCSS)](https://sass-lang.com/), [Bootstrap 5](https://getbootstrap.com/)
- **Animations**: [GSAP](https://gsap.com/)
- **UI Components**: [React-Bootstrap](https://react-bootstrap.github.io/), [Swiper](https://swiperjs.com/)
- **Icons**: [React Icons](https://react-icons.github.io/react-icons/)

## Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine.

### Installation

1.  Clone the repository:
    ```bash
    git clone <repository-url>
    ```
2.  Navigate to the project directory:
    ```bash
    cd sentratech-landing-page
    ```
3.  Install dependencies:
    ```bash
    npm install
    ```

### Running the Project

Start the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

### Building for Production

To create an optimized production build:

```bash
npm run build
```

To start the production server:

```bash
npm start
```

## Project Structure

- `src/app`: Contains the App Router pages and layouts.
- `src/components`: Reusable UI components (Hero, Header, Footer, etc.).
- `src/style`: Global styles, mixins, and component-specific SCSS files.
- `public`: Static assets like images and icons.

## Scripts

- `npm run dev`: Starts the development server.
- `npm run build`: Builds the application for production.
- `npm start`: Runs the built application in production mode.
- `npm run lint`: Runs ESLint to identify code issues.
